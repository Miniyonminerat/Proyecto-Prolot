# 🧪 Prolot

Prolot es un proyecto en desarrollo enfocado en la creación de una aplicación web funcional, construido con tecnologías frontend puras.

---

## 🚧 Estado del proyecto

⚠️ Actualmente en pausa  
🔄 Será retomado próximamente con mejoras estructurales y nuevas funcionalidades

---

## 🎯 Objetivo

El objetivo de Prolot es servir como un proyecto de práctica avanzada para:

- Mejorar habilidades en JavaScript  
- Aplicar lógica de programación real  
- Construir una aplicación completa desde cero  

---

## 🧠 Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## ⚙️ Funcionalidades (planeadas / en desarrollo)

- Interfaz dinámica  
- Manejo de datos en el frontend  
- Lógica interactiva avanzada  
- Posible integración con APIs
